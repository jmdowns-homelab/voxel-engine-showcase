# Voxel Engine Documentation

## Overview
This directory contains the documentation for the Voxel Engine, organized by category and purpose.

## Structure

### Domain Flows (`/domain_flows/`)
High-level descriptions of system interactions and data flows.
- **User Interaction Flows**: Player-initiated actions and UI interactions
- **System Flows**: Automated processes and background tasks

### Architecture (`/architecture/`)
Detailed documentation of system architecture and design decisions.

## Getting Started
1. Start with the [Architecture Overview](./architecture/overview.md)
2. Explore relevant domain flows for your area of interest
3. Refer to inline code documentation for implementation details
